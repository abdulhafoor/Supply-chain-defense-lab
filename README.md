# Supply Chain Defense Lab

A containerized cybersecurity lab simulating supply chain attacks and defenses.

## 🧪 Lab Components
- **Attacker**: injects tampered packages
- **Victim**: consumes packages
- **Monitor**: detects anomalies and sends alerts
- **Frontend**: dashboard (see [supply-chain-dashboard](https://github.com/abdulhafoor/Supply-chain-dashboard))
